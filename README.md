# installation



## requirements

- raspberry pi
- python3-dev
- pygame
- pydub
- gpio
- spidev
- fswebcam

## terminal

```bash
git clone https://github.com/jumang4423/jungle98.git
cd jungle98
sudo pip3 install pygame pydub spidev
python3 jungle98.py
```
